# InkReel Demo 视频脚本（3-4 分钟）

## 1. 开场（15s）— 产品页
- 打开 http://localhost:8766 → 展示 Landing 页面
- 口播："InkReel，一款 AI 驱动的小说转剧本工具。上传小说，自动识别人物，逐章转换为结构化 YAML 剧本。"

## 2. 登录注册（20s）— 认证
- 点击"开始使用" → 弹出登录弹窗
- 切换到"注册" → 输入邮箱 + 密码 → 发送验证码 → 注册成功自动跳转工具页
- 口播："支持邮箱注册登录，验证码保护。默认账户 admin@inkreel.local。"

## 3. 上传小说（30s）— 核心流程
- 点击"导入" → 选择 sample/sample_novel.txt
- 展示预览弹窗：检测到 3 章、武侠类型
- 确认导入 → 仓库自动刷新
- 口播："支持 .txt .md .docx .epub，自动检测章节和体裁。12 种正则覆盖中英文和罗马数字章节。"

## 4. 逐章转换（45s）— AI 核心能力
- 点击仓库中刚导入的小说 → 右侧显示章节列表
- 点击第 1 章"🎬 转换" → 等待 → YAML 场景出现在中栏
- 口播："三遍流水线：先提取人物，再逐章转换，注入前文摘要保持剧情连贯。"
- 快速点第 2 章、第 3 章

## 5. 合并导出（30s）— 产出
- 3 章完成后 → 底部出现"🧩 合并为完整剧本"
- 点击合并 → 完整 YAML 剧本呈现（人物表 + 场景 + 四幕结构）
- 点击"↓ 下载" → 保存到本地
- 口播："自动去重人物、重编号场景、推断四幕结构。一键下载 YAML 文件。"

## 6. 亮点功能（30s）— 加分项
- 切换右上角 🌐云端 → 🏠离线："支持云端 Mimo API 和本地 Ollama 双模式"
- 创建新改编："同一小说可创建多个改编版本，跨剧本复用已完成章节"
- 在线编辑 YAML："直接在浏览器编辑剧本，自动保存"

## 7. 结尾（10s）
- 口播："InkReel，让每本小说都有机会成为剧本。谢谢观看。"

---

## 录制建议
- OBS / Bandicam 录制 1920x1080
- 语速中速，每步操作完再说话
- 转换等待时可快进或剪辑
